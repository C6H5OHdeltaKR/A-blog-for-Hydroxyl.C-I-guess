# 第一篇文章：博客搭建记录

2024-05-29

## 为什么搭建博客

拥有一个个人博客可以：

- 记录学习过程，加深理解
- 分享知识，帮助他人
- 建立个人品牌
- 作为作品集展示

## 技术选择

本次使用 **VitePress** 搭建静态博客，主要优点：

1. **简单易用** - 只需 Markdown，无需复杂代码
2. **性能优秀** - 基于 Vite，构建速度快
3. **主题美观** - 默认主题简洁专业
4. **易于部署** - 可免费部署到 GitHub Pages

## Markdown 基础语法

### 标题

使用 `#` 符号表示标题，数量代表层级：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **粗体** 使用 `**文字**`
- *斜体* 使用 `*文字*`
- `代码` 使用反引号

### 列表

无序列表使用 `-` 或 `*`：

- 项目一
- 项目二

有序列表使用数字：

1. 第一步
2. 第二步

### 代码块

使用三个反引号包裹：

```javascript
function hello() {
  console.log('Hello, World!')
}
```

### 链接与图片

```markdown
[链接文字](https://example.com)
![图片描述](/path/to/image.png)
```
